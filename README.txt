Attributor is a small utility program to quickly add attribution text to an
image.


======================
==     RUNNING      ==
======================

This isn't currently published anywhere, but it is easy enough to download the
repository and run the attributor.py script as you would expect.

Make sure that any dependencies (tracked in requirements.txt) are installed
first.

The fonts/ directory is required and it must be in the same directory as the
attributor.py script - so if you just want to copy the files to your computer,
be sure to include the fonts too.


======================
==      USAGE       ==
======================

Attributor is a pretty simple program, and everything you really need to know
can be found in the help dialogue, accessed with

    $ attributor {-h|--help}
